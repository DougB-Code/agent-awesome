= Deploy Cloudflare Pilot
:description: Operations deployment commands for the Cloudflare pilot.

Run deployment commands from `deploy/cloudflare/worker`.
Use Node.js 22 or newer for Wrangler.
Make sure Docker is running before `wrangler deploy`; Wrangler builds and pushes the configured Container image during deployment.

== Install And Deploy

Generate the two internal bearer tokens before setting Worker secrets:

[source,sh]
----
GATEWAY_TOKEN="$(openssl rand -base64 32 | tr '+/' '-_' | tr -d '=')"
PERSISTENCE_TOKEN="$(openssl rand -base64 32 | tr '+/' '-_' | tr -d '=')"
----

[source,sh]
----
npm install --cache ../../../build/npm-cache
npm test
npx wrangler r2 bucket create agent-awesome-context
printf '%s\n' "$GATEWAY_TOKEN" | npx wrangler secret put AGENTAWESOME_GATEWAY_TOKEN
printf '%s\n' "$PERSISTENCE_TOKEN" | npx wrangler secret put AGENTAWESOME_PERSISTENCE_TOKEN
npx wrangler secret put SLACK_SIGNING_SECRET
npx wrangler secret put SLACK_BOT_TOKEN
npx wrangler secret put OPENAI_API_KEY
npx wrangler deploy
----

If you use an API token instead of `wrangler login`, export these values before the Wrangler commands:

[source,sh]
----
export CLOUDFLARE_ACCOUNT_ID=<account-id>
export CLOUDFLARE_API_TOKEN=<token>
----

After first deployment, set the Slack Events API request URL to:

[source,text]
----
https://agent-awesome.com/slack/events
----

Set `SLACK_ALLOWED_TEAM_ID` and `SLACK_ALLOWED_USER_ID` in `wrangler.jsonc` before redeploying a single-user pilot.
Leave `SLACK_ALLOWED_CHANNEL_ID` empty for direct-message testing.

The public `workers.dev` URL is disabled.
The configured production route is `agent-awesome.com/*`.

The `npm test` smoke test typechecks and bundles the Worker, verifies the
Container and Durable Object declarations in `wrangler.jsonc`, confirms
`/healthz` reaches the gateway, confirms `/mcp` is forwarded only with
`AGENTAWESOME_GATEWAY_TOKEN`, and confirms signed Slack Events API requests
reach the gateway path.

After deploy, wait several minutes for the first Container rollout, then inspect status:

[source,sh]
----
npx wrangler containers list
npx wrangler containers images list
----
