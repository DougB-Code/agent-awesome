= Cloudflare Persistence
:description: Cloudflare pilot persistence behavior.

The bundled memory database lives under `/app/data` inside the container.
Container disk is ephemeral after the instance sleeps, so durable context depends on snapshot restore and save behavior.

== Snapshot Storage

Memory context is restored from and saved to an R2 bucket through a private Worker endpoint.
The endpoint requires `AGENTAWESOME_PERSISTENCE_TOKEN` and is intended only for the colocated container.

== Operational Note

The current snapshot path is enough for an HTTP pilot while the container stays alive and snapshots are healthy.
Use a persistent memory backend before relying on the deployment for durable long-term personal memory.
