= Harness Runtime
:description: Harness runtime modes and credential commands.

The harness runs ADK web/API runtime modes for UI and gateway traffic.

== Web/API Mode

[source,sh]
----
go run ./cmd/agent-awesome run -- web --port 8080 api
----

Use web/API mode when the UI or gateway needs an assistant endpoint.

== Credentials

Credentials can be stored through the harness CLI:

[source,sh]
----
go run ./cmd/agent-awesome credentials set OPENAI_API_KEY
go run ./cmd/agent-awesome credentials remove OPENAI_API_KEY
----

Credential lookup should stay outside model provider implementations.
Provider adapters receive already selected and validated configuration from the model factory.
