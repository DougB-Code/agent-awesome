= Gateway
:description: Agent Awesome gateway subsystem documentation.

The gateway is the Go module in `gateway`.
It is a small HTTP command plane that fronts the harness, memory service, and channel adapters.

== Responsibilities

* Serve gateway health and sanitized status endpoints.
* Proxy assistant UI traffic to the harness.
* Proxy harness context tool traffic through authenticated `/api/context/*` routes.
* Proxy memory MCP traffic to the configured memory service.
* Optionally apply operator-provided runtime policy text to agent runs outside the generic proxy layer.
* Receive Slack events through HTTP Events API or Socket Mode.
* Optionally supervise sibling harness and memory processes for local deployments.

The gateway should not implement model provider behavior, memory storage, or Flutter UI state.

== API Surface

[cols="1,2",options="header"]
|===
|Route |Use

|`GET /healthz`
|Gateway liveness.

|`GET /api/gateway/status`
|Sanitized gateway, dependency status, and readiness.

|`GET /api/gateway/channels`
|Active and planned channel adapters.

|`POST /slack/events`
|Slack Events API ingress.

|`POST /mcp`
|UI-facing memory MCP proxy.

|`/api/context/*`
|Harness context tool proxy.

|`/api/*`
|Harness API proxy.
|===

The gateway keeps `/healthz` available during dependency startup.
Routes that need harness or memory readiness return `503 dependency not ready` until the relevant supervised service is connected.

Beta clients should use context through the gateway only. If the harness context
API is bound outside loopback, configure the same
`AGENTAWESOME_CONTEXT_API_TOKEN` / `--context-api-token` value on the harness
and gateway so the gateway can authenticate upstream requests.

`AGENTAWESOME_RUNTIME_POLICY_TEXT` is empty by default.
When explicitly configured by an operator, its text is injected into `POST /api/run_sse` user message parts with the `AGENT_AWESOME_RUNTIME_POLICY` marker.
Stable tool-call behavior belongs in the harness agent instructions and runtime callbacks, not gateway prompt text.
